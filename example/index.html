jQuery plugin - lazyloader example

sample code

javascript

    $(".img-wrap img").lazyloader({
        effect : "fadeIn",
        threshold : 100,
        imgSrcAttr : 'imgsrc',
        beforeLoadCls: 'loading'
    });
